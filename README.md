## What's Inside

This repository is a carefully curated collection of high-quality educational materials covering:

- **Data Structures & Algorithms** - Complete handwritten notes, learning plans, and problem-solving strategies
- **Programming Languages** - In-depth notes for Python, Java, C++, JavaScript, PHP, and more
- **Web Development** - HTML, CSS, ReactJS, NodeJS comprehensive guides
- **Database & SQL** - DBMS concepts, SQL queries, and operators
- **Interview Preparation** - Company-specific questions, TCS NQT prep, and programming QnA
- **Computer Science Fundamentals** - Operating Systems, Computer Networks, OOP concepts
- **Machine Learning** - Complete ML handwritten notes
- **Mobile Development** - Android development short notes
- **Resume Building** - ATS-friendly resume templates

## Featured Learning Plans

This repository includes structured learning roadmaps to help you master technologies systematically:

- **30 Days DSA Learning Plan** - Master Data Structures & Algorithms in one month
- **30 Days Python Learning Plan** - Complete Python from basics to advanced
- **45 Days Java Learning Plan** - Comprehensive Java programming journey
- **45 Days Web Development Learning Plan** - Full-stack web development roadmap

## Quick Start Guide

### For Beginners
1. Start with **C/C++ Handwritten Notes** for programming fundamentals
2. Move to **DSA** and follow the 30-day learning plan
3. Pick a language (Python/Java) and follow its structured roadmap
4. Practice with **Interview** materials

### For Interview Preparation
1. Review **Complete DSA Handwritten Notes**
2. Study **All Programming Interview QNA**
3. Practice **TCS NQT Questions** (for freshers)
4. Prepare your resume using **ATS Friendly Resume Template**

### For Web Developers
1. Start with **HTML Deep Dive Notes**
2. Progress to **JavaScript Notes**
3. Learn **ReactJS** for frontend
4. Master **NodeJS** for backend
5. Follow the **45 Days Web Development Learning Plan**
